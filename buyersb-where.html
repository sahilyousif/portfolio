BUYERS B-WHERE

Buyers B-Where is an online property searching tool to help current and future home owners, real estate agents and developers understand the risks of natural and human-health related hazards.

DESCRIPTION

Buyers B-Where is a project I've been working on since January 2016 with the Deparment of Urban Planning and Design at the University of Washington, Seattle. The goal of this project is to develop a web-based tool that will enable users to search for specific properties and receive a visual and statistical report describing its vulnerability to natural and human-health related hazards.

I created a minimal viable product to validate this idea with it's potential users. The prototype currently allows you to search for properties located in Galveston and Harris counties in Texas. The tool is built using JQuery and Leaflet, and queries real housing data from Zillow, Google Maps API and TAMU's Risk and Hazard Database. The application is hosted using Amazon Web Services.
