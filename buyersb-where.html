BUYERS B-WHERE

Buyers B-where is an online home searching tool to increase the understanding of natural and human-made hazards among current and future home owners, real estate agents, builders, and developers.

DESCRIPTION

This is a project I've been working on since January 2016 with the Deparment of Urban Planning and Design at the University of Washington Seattle. The goal of this project is to develop a web-based tool that will enable users to search for a specific land parcel and then receive a visual and statistical report describing its vulnerability to natural and human-health related hazards.

I created a Minimal Viable Product to validate this idea with potential users and investors. It's built using JQuery and Leaflet and queries real housing data from Zillow and TAMU's Risk Database. The application is hosted on Amazon Web Services. I created two versions to test different visualization styles.
